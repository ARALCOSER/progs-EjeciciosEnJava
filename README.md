Programa: Reserva de un asiento en sala de cine

Caso: 
Una sala de cine pequeña tiene 12 asientos organizados en 3 filas x 4 columnas.
Se requiere un programa que permita reservar un asiento y luego mostrar el estado de la sala.

Representación: 
0 = asiento libre
1 = asiento reservado

Lo que hacer el programa
- Crear una matriz 3x4 llamada asientos.
- Pedir al usuario la fila y la columna del asiento a reservar.
- Marcar el asiento como reservado.
- Mostrar la matriz completa en formato tabla usando bucles anidados.
- Valida el rango de la matriz y el ingreso que realiza el usuario para la reserva
